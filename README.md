明想 vs 冥想

\


在英文里只有一个词就是 Meditation，中文一般的翻译为冥想。语言文字都带着能量，冥想这个“冥”让人联想到冥间，幽暗不明，有种阴郁的感觉

\


而 Meditation 是个能带走阴郁带来愉悦、祥和，非常正能量的工具，所以我研习明想的大师就将其翻译为“日月明”的明想，就是带着光、让人内心更容易接受的名字。
